1. 基本类型

1.1. 原始类型 (primitive)

原始类的数据，是指值本身无法被改变，比如 JavaScript 中的字符串就是原始类型，如果你修改了 JavaScript 中字符串的值，那么 V8 会返回给你一个新的字符串，原始字符串并没有被改变，我们称这些类型的值为 "原始值"

JavaScript 中的原始值主要包括 null、undefined、boolean、number、string、bigint、symbol 这七种
