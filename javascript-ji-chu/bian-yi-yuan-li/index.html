1.2. TL;DR

V8 执行一段 JavaScript 代码所经历的主要流程了，这包括了：

  • 初始化基础环境
  • 解析源码生成 AST 和作用域
  • 依据 AST 和作用域生成字节码
  • 解释执行字节码
  • 监听热点代码
  • 优化热点代码为二进制的机器代码
  • 反优化生成的二进制机器代码
